# Rain

A simple calming rain simulation built with HTML5.

## Description

Rain is a minimalist web application that simulates the soothing sound and visuals of rainfall. Perfect for relaxation, meditation, or as a calming background while working or studying.

## Features

- HTML5-based rain simulation
- Calming visuals and sound effects
- Lightweight and easy to use

## How to Use

1. Open the application in your web browser
2. Sit back, relax, and enjoy the calming rain simulation

## Technologies

- HTML5
- CSS
- JavaScript
